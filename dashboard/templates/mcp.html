MCP Configuration

Add agentdx to your Claude Code or Cursor MCP configuration:

{
  "mcpServers": {
    "agentdx": {
      "command": "agentdx",
      "args": ["serve"]
    }
  }
}

Usage Examples

Search for code

// Using agentdx_search tool
{
  "query": "user authentication flow",
  "limit": 10
}

Find callers of a function

// Using agentdx_trace_callers tool
{
  "symbol": "handleLogin"
}

List Go files

// Using agentdx_files tool
{
  "pattern": "*.go",
  "limit": 50
}
